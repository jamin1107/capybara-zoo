## 一、需求概述

用户长按快门键5秒后，系统自动从SD卡中读取最近一次拍摄并保存的图像，重新执行裁剪、亮度/对比度增强后送入刷写队列，实现"重印上一张"功能。

### 1.1 触发条件

| 条件项 | 要求 |
|--------|------|
| 按键 | 快门键 |
| 动作 | 长按 ≥ 5秒 |
| 系统状态 | 设备空闲（无拍摄任务、无刷写任务、无排队任务） |
| 相纸状态 | 有相纸且可刷写 |
| SD卡状态 | 存在且包含至少一张历史照片 |

**"设备空闲"定义**：当前无任何图像采集、无任何刷写任务在执行、打印队列为空。

### 1.2 行为流程

```
[设备空闲状态]
     ↓
[长按快门 5秒]
     ↓
[检测相纸可用性] → 无相纸 → 语音提示"无相纸"，拒绝执行
     ↓ 有相纸
[检测SD卡及历史照片] → 无历史照片 → 语音提示"无历史照片"
     ↓ 有照片
[读取最新一张照片]
     ↓
[执行裁剪（匹配相纸比例）]
     ↓
[执行亮度/对比度增强]
     ↓
[送入刷写队列] → 语音提示"开始打印上一张"
     ↓
[刷写完成后] → 语音提示"打印完成"
```

## 八、备注

1. **"设备空闲"定义**：当前无任何图像采集、无任何刷写任务在执行、打印队列为空。
2. **长按判定**：需严格验证按压时长从第1秒到第5秒的累计时间。
3. **历史照片**：特指SD卡中最近一次拍摄并保存的图像。
4. **相纸检测时机**：相纸检测在长按5秒后进行，若此时无相纸则提示"无相纸"。
5. **异常处理优先级**：相纸拔出 > 历史照片损坏 > 新任务抢占
