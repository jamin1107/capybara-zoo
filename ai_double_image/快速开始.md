## 🚀 快速启动（三种方式）

---

### ✅ 方式一：纯前端预览模式（推荐！不需要安装任何依赖）

**最简单，直接可用！**

```bash
cd ai_double_image
./preview_frontend_only.sh
```

启动成功后，在浏览器打开：**http://localhost:8080**

**特点：**
- ✅ 不需要安装任何Python依赖
- ✅ 使用Canvas进行前端模拟融合（左右拼接）
- ✅ 完整测试所有UI交互
- ✅ 支持本地图片上传
- ✅ 支持结果保存

---

### 🟡 方式二：简化版Python服务器

如果已安装Python但不想安装依赖：

```bash
cd ai_double_image
python3 start_with_mock_backend.py
```

启动成功后，在浏览器打开：**http://localhost:3000**

**特点：**
- ✅ 只需要Python标准库
- ✅ 如果已安装Pillow，会进行左右拼接
- ✅ 如果未安装Pillow，直接返回第一张图片

---

### 🔴 方式三：完整版Flask服务器

需要安装Python依赖（推荐网络好时使用）：

```bash
cd ai_double_image
chmod +x start.sh
./start.sh
```

或手动启动：

```bash
cd ai_double_image/backend

# 安装依赖
pip3 install flask flask-cors pillow requests

# 启动服务器
python3 server.py
```

启动成功后，在浏览器打开：**http://localhost:3000**

**特点：**
- ✅ 完整支持OpenAI DALL-E API
- ✅ 完整支持Stability AI API
- ✅ 真实AI融合效果

## 🎨 使用说明

### 基本使用（模拟模式）

1. 打开浏览器访问 http://localhost:3000
2. 点击第一张图片卡片 → 选择本地图片
3. 点击第二张图片卡片 → 选择本地图片
4. AI模型选择"模拟融合（无需API）"
5. 点击"✨ 开始魔法融合"
6. 等待融合完成 → 查看结果
7. 点击"保持到本地"保存结果

### 使用真实AI API

#### OpenAI DALL-E
1. 设置环境变量：
   ```bash
   export OPENAI_API_KEY=你的OpenAI密钥
   ```
2. 重启后端服务器
3. AI模型选择"OpenAI DALL-E"
4. 输入融合提示词（可选）
5. 开始融合！

#### Stability AI
1. 设置环境变量：
   ```bash
   export STABILITY_API_KEY=你的Stability密钥
   ```
2. 重启后端服务器
3. AI模型选择"Stability AI"
4. 输入融合提示词（可选）
5. 开始融合！
